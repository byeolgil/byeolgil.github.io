# 프로젝트 개요

이 프로젝트는 장례 서비스 회사인 "별길상조"의 정적인 단일 페이지 웹사이트입니다. 웹사이트의 목적은 서비스에 대한 정보를 제공하고, 예상 비용 견적 양식을 제공하며, 고객 리뷰 및 연락처 정보를 표시하는 것입니다.

전체 웹사이트는 단일 `index.html` 파일에 포함되어 있습니다.

# 주요 파일

*   `index.html`: 웹사이트의 유일한 메인 파일입니다. HTML 구조, 콘텐츠 및 모든 CSS 스타일이 포함되어 있습니다. 콘텐츠는 한국어로 되어 있습니다.
*   `README.md`: 프로젝트 제목이 포함되어 있습니다.

# 빌드 및 실행

이 프로젝트는 정적 웹사이트이므로 빌드 프로세스가 없습니다. 웹사이트를 보려면 웹 브라우저에서 `index.html` 파일을 열면 됩니다.

# 개발 규칙

*   모든 스타일은 `index.html` 파일의 `<head>`에 있는 `<style>` 블록에 포함되어 있습니다.
*   프로젝트는 테마 (색상, 글꼴)에 CSS 변수를 사용합니다.
*   레이아웃은 반응형이며, 다양한 화면 크기에 대한 미디어 쿼리를 사용합니다.
*   이 프로젝트에는 JavaScript가 사용되지 않습니다.
